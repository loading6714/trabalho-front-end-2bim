# Imagens necessárias

O projeto usa apenas duas imagens próprias, referenciadas no HTML. Coloque os arquivos dentro da pasta `img/` com exatamente esses nomes:

- **motorista.png** → foto usada na seção "Sobre o motorista"
- **corolla.png** → foto do veículo (Toyota Corolla Altis Hybrid), usada na seção Hero
